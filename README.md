# Tetris – Classic Web Edition

A simple browser-based Tetris clone built with **HTML**, **CSS**, and **vanilla JavaScript** as a self-study project.

This project features a small landing page that describes the game and a separate game screen where you can actually play Tetris in your browser.

## How to Play (Local)

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. On the landing page, click **Play** to go to the game screen.
4. Press the **START** button in the middle of the board to begin.

### Keyboard Controls

- **← / →** – Move piece left / right  
- **↓** – Soft drop (faster fall)  
- **↑** or **Space** – Rotate piece  
- **R** – Restart after game over

## Features

- Classic 10×20 Tetris board
- Next piece preview
- Score system that rewards clearing multiple lines
- Smooth gameplay using the browser’s animation loop
- Simple modern landing page that introduces the game

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
